## Setup

#### Clone

`git clone git@github.com:alisd23/lazy-isomorphic-react.git`


#### Install dependencies

`npm install`


#### Development

`grunt serve:dev`


#### Production

`grunt build`  
`grunt serve:prod`


#### View Application

Go to `localhost:9999` to see the application
